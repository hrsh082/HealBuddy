# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-11

### Added
- Initial project setup with MERN stack
- User authentication system (Register/Login)
- Patient dashboard with symptom overview
- Symptom management module
  - 10 predefined symptoms
  - Severity tracking
  - Duration and date management
  - Status updates (Active/Resolved/Monitoring)
- Quick symptom solutions and immediate aid
- AI Chatbot integration
  - Conversation management
  - Symptom-specific responses
  - Conversation history
- Medical Report Scanner
  - File upload (PDF, JPG, PNG)
  - Report type classification
  - AI analysis framework
  - Summary and findings extraction
- User Profile Management
  - Personal information
  - Medical history
  - Medications tracking
- Database models (User, Symptom, Report, ChatMessage)
- RESTful API endpoints
- Frontend pages
  - Login/Register
  - Dashboard
  - Symptom Details
  - Report Upload
  - Chatbot
  - Profile
- Responsive UI with Tailwind CSS
- Form validation
- Error handling
- Toast notifications
